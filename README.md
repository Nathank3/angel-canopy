# Angels Regional Canopy Enterprises (ARCE)

Official landing page for Angels Regional Canopy Enterprises, featuring a
"Crystalline Canopy" glassmorphism aesthetic.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS** (v4 alpha / PostCSS)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Setup & Running

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features

- **GlassLoader**: Custom initial loading animation with brand logo.
- **Responsive Design**: Mobile-first approach with Tailwind.
- **Glassmorphism**: Extensive use of `backdrop-filter` and transparent
  gradients.
- **SEO Optimized**: Meta tags, Open Graph, Sitemap, and JSON-LD implemented.
- **Secure**: Content Security Policy (CSP) and secure external links.
